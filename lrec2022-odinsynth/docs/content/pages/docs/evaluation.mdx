# Evaluation

## Extrinsic

### TACRED 
Evaluating on [TACRED](https://nlp.stanford.edu/projects/tacred/) is available using `TacredEvaluation`. The rules are generated using `TacredRuleGeneration` over clusters of the initial data split. The clusters can be created using `tacred_data_prep.py`.
